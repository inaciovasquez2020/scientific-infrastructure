# Freeze Policy

This repository is a canonical template and standards surface.

Allowed:
- Template improvements
- Verification-surface hardening
- Documentation clarifications
- Non-breaking workflow updates

Forbidden:
- Repository-level claims stronger than template/standards scope supports
- Manuscript-specific completion claims
- Tracked transient build artifacts
- Undocumented canonical-surface mutations

Release rule:
- Any freeze tag must correspond to a passing canonical-surface test state.
