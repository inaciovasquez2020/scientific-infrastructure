# Proof Status

| Field | Value |
|---|---|
| Claim surface | Scientific Infrastructure |
| Status | Executable infrastructure layer. |
| Scope | Canonical reproducibility, provenance, CI, manifests, and executable verification layer for the URF ecosystem. |
| Non-claim | This repository is not the primary location for theorem-level URF claims. |

## Status discipline
- `Proved` = theorem-level claim closed inside declared scope.
- `Conditional` = depends on a named external hypothesis or unresolved lemma.
- `Executable` = build/test/verification surface closed without implying full theorem-level closure.
- `Experimental` = computational or exploratory artifact only.
